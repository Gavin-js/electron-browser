# kk-browser

基于[!url=https://github.com/pfrazee/electron-browser]升级,添加打包配置

**Install instructions**

To run this code, follow these steps:

```
git clone ssh://git@192.168.1.23:10022/Front-end/kk-browser.git
cd kk-browser
npm install
npm start
```
